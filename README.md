## What It Does
An AI-assisted web app that suggests recipes based on  ingredients you already have at home — reducing food  waste and simplifying everyday meal planning.

## Built With
- Lovable (AI-assisted frontend generation)
- React + Tailwind CSS

## Key Features
- Ingredient-based recipe discovery
- Step-by-step cooking instructions
- Clean, mobile-responsive UI

## Context
Built as part of GenAI & Design Thinking coursework at Masters' Union, Gurugram (2025–26).
Applied AI-assisted development to solve a real 
consumer problem end-to-end — from idea to live product.
